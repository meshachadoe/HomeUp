# HomeUp

## Running/Developing
### Setting up the repository
Clone the repo and install the dependencies in the frontend and the backend
```
git clone https://github.com/meshachadoe/HomeUp 
cd HomeUp/backend
npm install
cd ../client
npm install
```

### Running the project

To run the backend:
```
cd HomeUp/backend 
npm start
```

To run the frontend:
```
cd HomeUp/client
npm start
```
